# Site da Etuos

Site institucional da [Etuos](https://etuos.com), agência de marketing digital para brasileiros que empreendem nos Estados Unidos.

## Stack

- Next.js (App Router) + TypeScript + Tailwind CSS
- 100% estático (SSG), blog em MDX
- Deploy na Vercel

## Rodando localmente

```bash
npm install
npm run dev
```
